A responsive dashboard to browse, search, and sort country data. Built with React, TypeScript, and SCSS Modules.

## Features
*   **Search**: Real-time filtering by country name.
*   **Sort**: Order by Name or Population.
*   **Performance**: Implements infinite scroll and debouncing for efficiency.
*   **Style**: Custom modern UI with a "Summer Breeze" palette.

## How to Run

First, install the dependencies:

```bash
npm install
```

Then start the development server:

```bash
npm run dev
```

Open http://localhost:5173 to view it in the browser.

## Build for Production

To create an optimized build (or run via Docker):

```bash
npm run build
# or
docker build -t countries-app .
```
